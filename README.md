# Threat Intelligence Analyzer

A Python-based tool for analyzing the reputation of IP addresses and domains using Threat Intelligence APIs. The application queries external services, processes Indicators of Compromise (IOCs), and generates automated security reports.

## Features

- IP and domain reputation analysis
- Integration with VirusTotal API
- IOC (Indicators of Compromise) processing
- Automated report generation (TXT)
- Docker support for easy deployment
- Command-line interface (CLI)

## Use Cases

- Cyber Threat Intelligence (CTI)
- SOC (Security Operations Center) analysis
- OSINT investigations
- Security research and learning

##  Technologies

- Python
- Requests
- Dotenv
- Docker
